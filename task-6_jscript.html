Квадратне рівняння — це рівняння виду ax2 + bx + c = 0, де a не дорівнює 0.

Для розв'язання квадратного рівняння необхідно порахувати дискримінант

D = b2- 4ac.

Якщо D > 0, то рівняння має два різних дійсних кореня. Якщо D = 0, то обидва кореня дійсні і рівні. Якщо D < 0, то оба кореня комплексні числа.
Корені квадратного рівняння знаходяться за формулою
x1,2 =  -b ± √D
2a

Якщо в вашому квадратному рівнянні є віднімання, то перед відповідними коефіцієнтами в калькуляторі необхідно поставити знак мінус ("-"). Наприклад: x2 - 4x - 5 = 0, коефіцієнти 1, -4, -5.
Якщо в вашому квадратному рівнянні менше трьох доданків, то поруч з відсутніми доданками в калькуляторі поставте коефіцієнт нуль ("0"). Наприклад: x2 - 4x = 0, коефіцієнти 1, -4, 0.
